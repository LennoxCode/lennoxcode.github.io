Wing IT

Wing IT! Is a two-player local coop game in which two birds try to hone the ball to the other side and thus gaining points. The game was created in unity as a part of course work where we had to create two complete games ourselves during the semester.

Wing IT! TD

WWing IT! TD a tower defense game inspired by Bloons. The player tries to fence off growing waves of enemies by recruiting and upgrading bird warriors. The game has five different towers which each offer two unique upgrade paths, two levels and three different types of enemies.

Caption Hook

Caption Hook is 2D jump & run made with the unity game engine. The player traverses’ various level by only using the ability of a hook shot. Each level increases in difficulty and represents a different era of the internet. This game was part of a group assignment for the course “advanced game programming”. The only direction and constraint given was that we had to create a physics game.

Number classification Neural Network

In the fifth semester I selected a machine learning course. In preparation I decided to implement a neural network from scratch in my spare time by only using python and the NumPy library for the required matrix multiplication. I used the MNIST dataset for training and after training the network boasts an accuracy of 97% for the test set.

Trading Game WIP

Trading Game is a strategy game which plays on a 3D hex grid map. I am developing this game in my spare time when I don’t have complete university assignments.
